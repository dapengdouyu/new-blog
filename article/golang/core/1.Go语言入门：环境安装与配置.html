一、Go语言安装 ​

1.1 系统要求 ​

Go语言支持以下操作系统：

  • Windows: Windows 7及以上版本
  • macOS: macOS 10.12及以上版本
  • Linux: 主流Linux发行版

1.2 Windows安装 ​

方式一：官方安装包（推荐） ​

  1. 下载安装包

    • 访问 Go官网
    • 下载Windows版本（如：go1.21.x.windows-amd64.msi）

  2. 运行安装程序

    • 双击.msi文件
    • 按照向导完成安装（默认安装到C:\Program Files\Go）

  3. 验证安装

    cmd
    go version
    1

    如果显示版本号，说明安装成功。

方式二：使用包管理器 ​

使用Chocolatey：

cmd
choco install golang
1

使用Scoop：

cmd
scoop install go

1.4 Linux安装 ​

二、环境变量配置 ​

2.1 基本环境变量 ​

Go语言需要配置以下环境变量：

变量名说明默认值
GOROOTGo的安装路径/usr/local/go (Linux/macOS) 或 C:\Program Files\Go (Windows)
GOPATH工作空间路径~/go (Linux/macOS) 或 %USERPROFILE%\go (Windows)
PATH可执行文件路径需要添加$GOROOT/bin或%GOROOT%\bin

2.2 Windows配置 ​

方法一：图形界面配置 ​

  1. 右键"此电脑" → "属性" → "高级系统设置"
  2. 点击"环境变量"
  3. 在"系统变量"中添加：
    • GOROOT: C:\Program Files\Go
    • GOPATH: C:\Users\你的用户名\go
  4. 在PATH中添加：%GOROOT%\bin

方法二：命令行配置（PowerShell） ​

powershell
# 设置GOROOT（通常不需要，安装程序已设置）
[Environment]::SetEnvironmentVariable("GOROOT", "C:\Program Files\Go", "User")

# 设置GOPATH
[Environment]::SetEnvironmentVariable("GOPATH", "$env:USERPROFILE\go", "User")

# 添加到PATH
$currentPath = [Environment]::GetEnvironmentVariable("Path", "User")
[Environment]::SetEnvironmentVariable("Path", "$currentPath;C:\Program Files\Go\bin", "User")

三、中国用户代理配置 ​

由于网络原因，中国用户访问Go官方源可能较慢，需要配置代理。

3.1 配置Go代理 ​

七、常见问题解决 ​

7.1 安装问题 ​

问题：go: command not found ​

解决方案：

  1. 检查Go是否已安装：which go（Linux/macOS）或where go（Windows）
  2. 检查PATH环境变量是否包含Go的bin目录
  3. 重新加载shell配置：source ~/.bashrc或重启终端

问题：权限错误（Linux/macOS） ​

解决方案：

bash
# 使用sudo安装到系统目录
sudo tar -C /usr/local -xzf go1.21.5.linux-amd64.tar.gz

# 或安装到用户目录
mkdir -p ~/go
tar -C ~/go -xzf go1.21.5.linux-amd64.tar.gz
export PATH=$PATH:~/go/go/bin

八、最佳实践 ​

8.1 环境配置建议 ​

  1. 使用官方安装包：最稳定可靠
  2. 配置代理：中国用户必须配置
  3. 使用版本管理工具：方便切换版本
  4. 定期更新：保持Go版本和工具最新

8.2 开发环境建议 ​

  1. 使用VSCode：功能强大，社区活跃
  2. 安装必要工具：gopls、goimports、golangci-lint
  3. 配置代码格式化：保持代码风格一致
  4. 启用实时检查：及时发现问题

8.3 项目结构建议 ​

myproject/
├── go.mod          # Go模块文件
├── go.sum          # 依赖校验文件
├── main.go         # 主程序
├── cmd/            # 可执行文件
├── internal/       # 内部包
├── pkg/            # 可导出包
└── .vscode/        # VSCode配置
    └── settings.json
