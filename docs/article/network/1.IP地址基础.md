# IP地址基础

## 一、什么是IP地址

IP地址（Internet Protocol Address）是互联网协议地址的简称，用于在网络中唯一标识一台设备。就像现实生活中的门牌号一样，IP地址让网络中的设备能够找到彼此并进行通信。

## 二、IP地址的版本

IP地址目前主要有两个版本：

### 2.1 IPv4

- **位数**：32位
- **表示方式**：点分十进制，如 `192.168.1.1`
- **地址数量**：约42亿个（2^32）
- **现状**：目前广泛使用，但地址已基本耗尽
- **示例**：`192.168.1.1`、`10.0.0.1`、`172.16.0.1`

### 2.2 IPv6

- **位数**：128位
- **表示方式**：冒号分隔的十六进制，如 `2001:0db8:85a3:0000:0000:8a2e:0370:7334`
- **地址数量**：几乎无限（2^128）
- **现状**：逐步推广中，是未来的趋势
- **示例**：`2001:0db8::1`、`fe80::1`

**注意**：本文主要讨论IPv4地址，因为它是目前最常用的版本。

## 三、IPv4地址的组成

### 3.1 基本结构

一个IPv4地址由32位二进制数组成，通常分为4段，每段8位（0-255），用点号分隔：

```
192.168.1.1
```

### 3.2 二进制表示

每个IPv4地址都可以转换为二进制：

```
十进制：192.168.1.1
二进制：11000000.10101000.00000001.00000001
        └───┬───┘└───┬───┘└───┬───┘└───┬───┘
          192      168        1        1
```

### 3.3 地址范围

- **最小值**：`0.0.0.0`（二进制：00000000.00000000.00000000.00000000）
- **最大值**：`255.255.255.255`（二进制：11111111.11111111.11111111.11111111）
- **总数量**：2^32 = 4,294,967,296 个地址

### 3.4 地址格式

IPv4地址的格式规则：
- 由4个数字组成，用点号（.）分隔
- 每个数字范围：0-255
- 不能有前导零（如 `192.168.001.001` 是错误的）

**正确示例**：
- ✅ `192.168.1.1`
- ✅ `10.0.0.1`
- ✅ `172.16.0.1`

**错误示例**：
- ❌ `192.168.001.001`（不能有前导零）
- ❌ `256.1.1.1`（超过255）
- ❌ `192.168.1`（缺少一段）

## 四、IP地址的作用

### 4.1 设备标识

IP地址就像设备的"身份证"，在网络中唯一标识一台设备：

```
设备A：192.168.1.100
设备B：192.168.1.101
设备C：192.168.1.102
```

### 4.2 网络通信

设备通过IP地址进行通信：

```
设备A（192.168.1.100） → 发送数据 → 设备B（192.168.1.101）
```

### 4.3 路由选择

路由器根据IP地址决定数据包的转发路径：

```
数据包：源IP（192.168.1.100） → 目标IP（10.0.0.1）
路由器：根据目标IP地址选择转发路径
```

## 五、常见IP地址示例

### 5.1 家庭网络

```
路由器：192.168.1.1
电脑：192.168.1.100
手机：192.168.1.101
```

### 5.2 企业网络

```
服务器：10.0.0.10
办公电脑：10.0.0.100
打印机：10.0.0.200
```

### 5.3 特殊地址

```
本地回环：127.0.0.1（localhost）
广播地址：255.255.255.255
```

## 七、总结

- IP地址是网络中设备的唯一标识
- IPv4是目前最常用的版本（32位）
- IPv6是未来的趋势（128位）
- IP地址由4个0-255的数字组成
- 理解IP地址是学习网络的基础
