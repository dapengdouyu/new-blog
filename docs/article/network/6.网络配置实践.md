# 网络配置实践

## 一、实际应用场景

### 1 家庭网络配置

典型的家庭网络配置：

```
路由器公网IP：由ISP分配，如 123.45.67.89
路由器内网IP：192.168.1.1（网关）
DHCP地址池：192.168.1.100 ~ 192.168.1.200
子网掩码：255.255.255.0（/24）
DNS服务器：192.168.1.1（路由器）或 8.8.8.8（Google DNS）
```

### 2 企业网络规划

大型企业可能使用：

```
总部网络：10.0.0.0/8
  - 部门A：10.1.0.0/16
  - 部门B：10.2.0.0/16
  - 服务器区：10.10.0.0/16
  - 办公区：10.20.0.0/16

分支机构：172.16.0.0/12
  - 分公司1：172.16.0.0/16
  - 分公司2：172.17.0.0/16
```

### 3 NAT（网络地址转换）

由于IPv4地址短缺，大多数家庭和企业使用NAT技术：

**工作原理**：
1. 内网设备使用私有IP地址（如 192.168.1.100）
2. 路由器拥有公网IP地址（如 123.45.67.89）
3. 当内网设备访问互联网时，路由器将私有IP转换为公网IP
4. 返回的数据包再由路由器转发给对应的内网设备

**优势**：
- 节省公网IP地址
- 提供一定的安全隔离
- 简化网络管理

## 二、如何查看和配置IP地址

### 1 Windows系统

**查看IP地址**：
```cmd
ipconfig
ipconfig /all
```

**配置IP地址**：
1. 打开"网络和共享中心"
2. 点击"更改适配器设置"
3. 右键点击网络适配器 → "属性"
4. 选择"Internet协议版本4（TCP/IPv4）" → "属性"
5. 选择"使用下面的IP地址"并填写

### 2 Linux/macOS系统

**查看IP地址**：
```bash
ifconfig
# 或
ip addr show
```

**配置IP地址**：
```bash
# 临时配置
sudo ifconfig eth0 192.168.1.100 netmask 255.255.255.0

# 永久配置（Ubuntu/Debian）
sudo nano /etc/netplan/01-netcfg.yaml
```

### 3 路由器配置

1. 登录路由器管理界面（通常是 192.168.1.1 或 192.168.0.1）
2. 进入"网络设置"或"LAN设置"
3. 配置IP地址池范围
4. 设置子网掩码和网关
