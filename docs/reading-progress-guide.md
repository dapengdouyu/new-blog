# 阅读进度功能指南

你的博客现在支持完整的阅读进度功能！

## 功能特性

### 📊 阅读进度条
- **位置**: 页面顶部显示细长的进度条
- **显示时机**: 滚动超过50px时出现
- **样式**: 渐变色，支持深色模式
- **动画**: 平滑过渡效果

### ⏱️ 阅读时间估计
- **自动计算**: 基于文章字数和阅读速度自动计算
- **默认速度**: 200字/分钟
- **显示位置**: 右侧悬浮区域

### 🔝 回到顶部按钮
- **智能显示**: 滚动超过200px时显示
- **平滑滚动**: 点击后平滑滚动到页面顶部
- **悬浮设计**: 现代化悬浮按钮样式

## 使用方法

### 基本使用

在任何 Markdown 文件中添加：

```markdown
<ReadingProgress />
```

### 高级配置

```markdown
<!-- 显示阅读时间，默认200字/分钟 -->
<ReadingProgress :showReadingTime="true" :wordsPerMinute="250" />

<!-- 隐藏阅读时间 -->
<ReadingProgress :showReadingTime="false" />

<!-- 隐藏回到顶部按钮 -->
<ReadingProgress :showScrollToTop="false" />
```

## 技术实现

### 性能优化
- **防抖更新**: 使用 `requestAnimationFrame` 优化滚动事件
- **精确计算**: 考虑视窗高度的准确进度计算
- **内存管理**: 正确的事件监听器清理

### 可访问性
- **语义化标签**: 使用适当的 ARIA 属性
- **键盘支持**: 支持键盘导航
- **屏幕阅读器**: 兼容屏幕阅读器

### 响应式设计
- **移动端优化**: 在小屏幕上隐藏某些元素
- **触摸友好**: 适合触摸设备的按钮尺寸
- **自适应布局**: 根据屏幕尺寸调整位置

## 自定义样式

组件使用了 CSS 变量，你可以通过修改 VitePress 主题来自定义样式：

```css
:root {
  --vp-c-brand: #3eaf7c;  /* 主色调 */
  --vp-c-brand-light: #4abf8a; /* 浅色调 */
  --vp-c-bg: #ffffff;    /* 背景色 */
  --vp-c-text-2: #6b7280; /* 次要文字 */
}
```

## 故障排除

### 进度条不显示
- 检查是否在页面顶部添加了 `<ReadingProgress />`
- 确认组件已正确注册到主题中

### 进度计算不准确
- 确保页面内容有足够的长度
- 检查是否有固定定位的元素影响计算

### 样式异常
- 确认 VitePress 版本兼容
- 检查自定义 CSS 是否覆盖了默认样式
